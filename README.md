# 📊 Personal Finance Dashboard

A data-driven personal finance dashboard built using **Python, Jupyter Notebooks, SQL, and interactive visualization tools** to analyze financial behavior using publicly available datasets.

---

## 🚀 Project Overview

This project transforms raw financial transaction data into actionable insights through an interactive dashboard.

Instead of relying on personal finance exports, this project uses a dataset sourced from Hugging Face, making it fully reproducible and suitable for data analysis and portfolio demonstration.

It helps answer key financial questions such as:

- Where does most spending occur?
- How do expenses evolve over time?
- What are the dominant spending categories?
- How does income compare against expenses?
- What trends can be identified in financial behavior?

---

## 🧱 Tech Stack

### 📌 Core Technologies
- Python  
- Jupyter Notebook (.ipynb) for exploratory analysis  
- SQL / Relational Database  

### 📊 Data & Visualization
- Pandas – data manipulation  
- Plotly – interactive charts  
- Streamlit – dashboard interface  

### ⚙️ Tools & Infrastructure
- SQLAlchemy – ORM & database connection  
- Docker & Docker Compose – containerization  
- Git & GitHub – version control

## 🎯 Motivation

This project focuses on building an **end-to-end data pipeline**, combining:

- Data ingestion from external sources  
- Data cleaning and transformation  
- Database design and querying
